# Contributing to PBLC

Thank you for your interest in contributing to PBLC!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/PBLC.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m "Add your message here"`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Create a Pull Request

## Development Setup

Please refer to [QUICKSTART.md](QUICKSTART.md) for detailed setup instructions.

## Code Standards

- Follow C# coding conventions
- Write clean, readable code
- Comment your code where necessary
- Test your changes before submitting

## Reporting Issues

If you find a bug or have a suggestion:
1. Check if the issue already exists
2. If not, create a new issue with a clear title and description
3. Include steps to reproduce (for bugs)
4. Include your environment details

## Pull Request Process

1. Ensure your code builds without errors
2. Update documentation if needed
3. Follow the existing code style
4. Wait for review and address any feedback

Thank you for contributing!
