# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- User authentication system with Admin, Teacher, and Student roles
- Course management system with enrollment features
- Lecture and notes management with file uploads
- Assignment creation, submission, and evaluation system
- Q&A forum with nested replies
- Performance tracking and analytics
- Admin control panel for system management
- Department-wise course organization
- Role-based dashboards and access control
- MySQL database integration

### Features
- Secure login and registration with ASP.NET Identity
- File upload support (PDF, PPT, DOC)
- Video lecture links support
- Assignment deadline management
- Grade tracking and reporting
- Thread-based Q&A discussions
- User activation/deactivation by admin

### Technical
- ASP.NET Core 9.0 MVC architecture
- Entity Framework Core 9.0
- MySQL with Pomelo provider
- Bootstrap 5 responsive UI
- Clean Architecture pattern (Core, Infrastructure, Services, Web)
